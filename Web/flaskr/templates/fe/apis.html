Anti-Buzzer API

Twitter

Pada saat ini, hanya tersedia data dari twitter. Namun, dikarenakan kebijakan penggunaan data dari twitter, penggunaan API ini terbatas pada 100.000 pemanggilan kumulatif. Kebihkan ini dapat membuat API bisa saja tidak beroperasi sewaktu-waktu. API anti buzzer dari data twitter menerima input berita screen name / id akun twitter. Setelah itu, API akan mengembalikan nilai probabilatas buzzer dari akun tersebut. Apabila terjadi kesalahan, api akan mengembalikan nilai kosong tanpa ada perubahan kode.

URL GET, POST

/api/buzzer

Parameters

# Nama Wajib Nilai Deskripsi
1 screnname wajib - Id akun twitter atau nama pengguna. Id akun twitter akan menjadi prioritas jika terdapat dua akun dengan nama yang sama. Jangan menggunakan tanda @ sebelum mengisi akun twitter.

Results

# Nama Tipe Deskripsi
1 screnname string Id akun twitter atau nama pengguna yang dikirimkan.
2 prob float Probabilitas jika akun adalah buzzer
